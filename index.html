01 - Boids simulation

This was a project I did for my artificial intelligence class in high school. I was learning react at the time, so I decided to use Vite, React with Typescript to make this project interactive. It's about simulating how birds like to flock together, but at the same time, don't want to stay too close to each other.

02 - R2 CLI Tool

This NPM package is a CLI tool that allows you to download, upload, list, and delete files from your Cloudflare R2 bucket. It uses AWS S3 SDK to interact with the R2 API. I suddenly came up with an idea for temporary cloud storage for sharing files, so I made this CLI tool to make it easier to manage files.

03 - Relay

This small project is a bare-bones URL shortening tool. It allows users to shorten extremely long URLs into shorter ones which are easier to remember and share. It's built with NextJS frontend and Rust backend, with a PostgreSQL instance to store the shortened URLs.
